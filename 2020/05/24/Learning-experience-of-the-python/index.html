Learning experience of the python

​ I started learning python about a month ago. The learning experience of basics of the software Visual Basic which is taught in the school makes me feel quite familiar with the grammar and the type of variables. To make the program more clearly and easier to run the program without checking it again and again we choose to use Jupyter Notebook to done the program.

​ Python is one of the most common computer languages. It has been used in many different areas, for example, the GUI development, some simple Program compilation and the scientific calculation. I want to do some data analysis by using the language and the python is the most suitable for me who is a green hand in the computer language learning. It would take me least time to done the most work when doing data analysis by using the python.

​ I found that the computer language learning is an interesting process, I could do some simple program during each period of learning. I was able to find out my progress by writing the program with more complex and longer logic. For example, I wrote a small process which set the Username and Password. At first, if you typed in wrong username or password, the program needed to be restarted and type in all the data again. While, through some learning you would able to type in your username wrongly, you would be asked to type it again, so do the password. If you have typed in wrong username or password for over 5 times you are unable to try again and not allowed entering the website.

​ Some interesting programs may be followed if it is possible.

​ Thank you for watching my blog.
