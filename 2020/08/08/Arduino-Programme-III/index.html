Arduino Programme III

This time, a new unit which is a button was added into our input module and used to control the 1 or 0 signal by using your finger.

We have written some simple code to make the button able to control the LED light.


#define LED 8

#define KEY 7

void setup() {

pinMode(LED,OUTPUT);

pinMode(KEY,INPUT);

}

}

void loop() {

digitalWrite(LED,LOW);

if(!digitalRead(KEY)){

digitalWrite(LED,HIGH);

}

else{

digitalWrite(LED,LOW);

}

}

Then, we try to use the button unit to control the buzzer.

It is interested that there is a seal for the speaker for the buzzer and teacher told us not to open it in order to control the volume of the buzzer. The teacher just opened part of the seal to show us, how loud the buzzer is. Wow, it is just like the sound of the emergency alarm.

The code we designed is shown below

#define BUZZ 8

#define KEY 7

void setup() {

pinMode(BUZZ,OUTPUT);

pinMode(KEY,INPUT);

}

void buzz(int times){

digitalWrite(BUZZ,LOW);

delay(times);

digitalWrite(BUZZ,HIGH);

delay(times);

}

void loop() {

digitalWrite(BUZZ,LOW);

if(!digitalRead(KEY)){

digitalWrite(BUZZ,HIGH);

}

else{

digitalWrite(BUZZ,LOW);

}

}
