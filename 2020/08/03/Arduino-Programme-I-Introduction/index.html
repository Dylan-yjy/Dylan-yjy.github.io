Introduction

What is the Arduino?

Arduino is an open source electronic prototype platform which comes from Italy.

It is a highly compatible development platform as it is able to work on different computer system as Windows, Mac OS，Linux.

What about the software to running and testing the programmes?

Some primary code are shown below for some simple progrmme

void setup() {

// initialize digital pin LED_BUILTIN as an output.

pinMode(LED_BUILTIN, OUTPUT);

}

// the loop function runs over and over again forever

void loop() {

digitalWrite(LED_BUILTIN, HIGH); // turn the LED on (HIGH is the voltage level)

delay(1000); // wait for a second

digitalWrite(LED_BUILTIN, LOW); // turn the LED off by making the voltage LOW

delay(1000); // wait for a second

}
