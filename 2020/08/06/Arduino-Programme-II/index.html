Arduino Programme II

We have done three small programmes during our second class.

(i). Firstly, we tried to make the equipment which would be able to displace the automatic light we use to light up the way we walk at the night.

An infrared sensor was used in the equipment as a core component.

We design the code as below

void setup() {

// initialize digital pin LED_BUILTIN as an output.

pinMode(8, OUTPUT);

pinMode(7, INPUT);

}

// the loop function runs over and over again forever

void loop() {

if(digitalRead(7)){

digitalWrite(8, HIGH); // turn the LED on (HIGH is the voltage level)

}

else{

digitalWrite(8, LOW); // turn the LED off by making the voltage LOW

When there is someone passes by the sensor the sensor will considered the signal as “1” and higher the voltage of the light to make it light up.

(ii). The second design we make has almost the same route to use in our daily life. However, we use a sound senor instead.

A sound sensor is shown below

We used almost the same code as the infrared sensor

void setup() {

// initialize digital pin LED_BUILTIN as an output.

pinMode(8, OUTPUT);

pinMode(7, INPUT);

}

// the loop function runs over and over again forever

void loop() {

if(digitalRead(7)){

digitalWrite(8, HIGH); // turn the LED on (HIGH is the voltage level)

delay(3000)

}

else{

digitalWrite(8, LOW); // turn the LED off by making the voltage LOW

However, “delay” was added in the code to ensure the light will maintain bright enough for 3 seconds (3000 means 3000 milliseconds)

(iii). Finally, we use temperature sensor to simulate the computer cooling fan.

Here shows the equipment I have made below

#include<OneWire.h>

#include<DallasTemperature.h>

#define BUS 7

OneWire onewire(BUS);

DallasTemperature senors(&onewire);

void setup(){

// put your setup code here, to run once:

Serial.begin(9600);

senors.begin();

pinMode(8,OUTPUT);

}

void loop(){

// put your main code here, to run repeatedly:

senors.requestTemperatures();

Serial.write(“温度：”);

Serial.print(senors.getTempCByIndex(0));

Serial.println(“摄氏度”);

digitalWrite(8,LOW);

if (senors.getTempCByIndex(0)>34){

digitalWrite(8,HIGH);

}

}
